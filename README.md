# ToDoリストのアプリ

vue 2.6.11  
@vue/cli 4.5.15  
データ保存先はLocal Strage  

**アプリ概要**  
メモの登録・更新・削除が出来るシンプルなアプリです。  
ローカル環境で使用可能です。

**使用方法**    
`clone`もしくは`fork`でローカルに落として以下をターミナルで実行。

```
npm run build
```

todo_list/dist/index.htmlをブラウザで開けば使用可能です！
